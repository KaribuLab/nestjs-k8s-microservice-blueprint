# Agents

## NestJS microservices

This agent is used to create a NestJS microservice. It will uses the NestJS microservices skill (`nestjs-microservices`) to create the microservice.

## Helm chart

This agent is used to create a helm chart and related resources for a NestJS microservice. It will uses the Helm Charts skill (`helm-charts`) to create the helm chart and related resources.

## Screaming Architecture

This agent is used to create a screaming architecture for a NestJS microservice. It will uses the Screaming Architecture skill (`screaming-architecture`) to create the screaming architecture.

## Owasp Security Check (OWASP)

This agent is used to create a owasp security check for a NestJS microservice. It will uses the Owasp Security Check skill (`owasp-security-check`) to create the owasp security check.
